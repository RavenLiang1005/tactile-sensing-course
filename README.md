# 触觉传感器和机器人灵巧操作

根据《触觉传感器和机器人灵巧操作》课程大纲制作的静态课程网站，包含：

- 6 次课程的核心知识点、课后任务和实践要求
- 每次课配套的论文、项目页、开源代码和 Demo
- 可筛选的开放资源库
- 响应式布局，支持手机与桌面浏览

## 本地预览

直接打开 `index.html`，或在目录中启动任意静态文件服务器：

```bash
python -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 部署到 GitHub Pages

1. 在 GitHub 新建仓库并推送本目录文件。
2. 进入仓库的 `Settings` → `Pages`。
3. 在 `Build and deployment` 中选择 `Deploy from a branch`。
4. 选择 `main` 分支和根目录 `/ (root)`，保存后等待部署完成。

## 内容维护

课程与资源数据集中在 `app.js` 的 `courses` 和 `resources` 数组中，可直接增删。

> 部分历史开源仓库已进入只读归档状态，但代码、文档和示例仍可用于教学。引用论文或代码时请遵循原项目许可。
